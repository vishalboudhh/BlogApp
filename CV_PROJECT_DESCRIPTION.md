## Detailed Version (For Portfolio/GitHub)

### BlogApp - Full-Stack Social Media Platform

A Twitter-like social blogging platform built with the MERN stack, featuring real-time updates, user authentication, and social interactions.

**Technologies Used:**
- **Frontend:** React 19, Vite, Redux Toolkit, React Router, Tailwind CSS, Axios
- **Backend:** Node.js, Express.js, MongoDB, Mongoose, JWT, bcryptjs, Multer
- **Authentication:** JWT tokens with httpOnly cookies, protected routes
- **Database:** MongoDB Atlas with Mongoose ODM
- **Deployment:** Railway (backend), Vercel (frontend)

**Key Features:**
- User authentication and authorization (signup, login, logout)
- Create, read, and delete posts/tweets
- Like/unlike functionality with real-time updates
- Follow/unfollow system for social connections
- Bookmark posts for later viewing
- User profiles with bio and profile picture uploads
- Two feed modes: "For You" (all posts) and "Following" (filtered feed)
- Protected routes and authentication middleware
- Responsive UI with Tailwind CSS

**Technical Highlights:**
- Implemented secure authentication using JWT tokens stored in httpOnly cookies
- Built RESTful API with proper error handling and validation
- Used Redux Toolkit for centralized state management
- Implemented custom React hooks for data fetching and authentication
- Configured CORS for cross-origin requests
- Set up file upload handling with Multer for profile pictures
- Designed scalable MongoDB schemas with proper relationships
- Implemented real-time UI updates using Redux state management

**Project Structure:**
- Modular backend architecture with separate controllers, routes, and middleware
- Component-based React frontend with reusable UI components
- Custom hooks for API calls and authentication logic
- Environment variable configuration for development and production
